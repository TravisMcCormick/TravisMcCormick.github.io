## ðŸŽ¨ Image Guidelines

### File Formats
- **Photos:** JPG (smaller file size, good for photos)
- **Graphics/Logos:** PNG (transparency support)
- **Patterns:** SVG (scalable) or PNG

### Optimization
- Keep file sizes reasonable (under 1MB per image)
- You can use online tools like TinyPNG or Squoosh to compress

### File Naming
- Use lowercase
- Use hyphens for spaces (not underscores)
- Be descriptive
